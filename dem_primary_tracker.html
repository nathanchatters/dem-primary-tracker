# Candidate Avg % Change2
  1. Sources. All figures are 270toWin's national polling average for the 2028 Democratic nomination, itself an aggregate of multiple pollsters. Dates before this tracker started running daily are backfilled: 270toWin doesn't publish a history of its own past averages, so those points are our average of that day's individual polls as listed on their page - an approximation of, not identical to, 270toWin's own methodology.
  2. Change. The difference versus the most recent earlier date this candidate has a value for - not a fixed window like "vs. yesterday" or a rolling average. Early history is sparse (dated to whenever a poll actually happened), so a "change" there can span weeks; once enough daily snapshots have accumulated going forward it converges to a day-over-day comparison.
  3. Update frequency. A GitHub Action re-scrapes 270toWin once a day at 14:00 BST.
